在线客服NLP处理
==

基于Flask、Keras（tensorflow）实现的问答机器人，可快速整合企业内业务数据，对外提供智能问答功能。

## 工程环境

* Python 3.6
* Pip 9.0.1 

## 工程结构

```
.
├── Dockerfile                          Docker构造文件
├── README.md                           文档
├── app.py                              web启动器
├── config                              静态配置数据
│   ├── dict                            自定义词典
│   └── stopword                        停用词
├── core                                nlp逻辑核心
│   ├── handle                          处理器包
│   └── manager.py                      处理器工厂
├── data                                数据持久化
├── depend                              依赖的外部接口
├── train                               深度学习训练工程
└── requirements.txt                    依赖文件
```

    初始化项目，在根目录是用`pip install -r requirements.txt`安装所有依赖
